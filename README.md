# ⚡️ GeniusCSS - Gerador de Design com IA

O **GeniusCSS** é uma ferramenta inteligente que transforma descrições em texto em componentes reais de HTML e CSS. Utilizando a API do Groq com o modelo Llama 3.3, o projeto permite materializar ideias de interface instantaneamente.

## 🚀 Tecnologias Utilizadas
- **HTML5 & CSS3**: Estrutura e estilização moderna com Grid e Flexbox.
- **JavaScript (ES6+)**: Manipulação de DOM e consumo de API assíncrona.
- **Groq Cloud API**: Processamento de linguagem natural com Llama 3.3.
- **Netlify**: Hospedagem e Deploy contínuo.

## 🛠️ Como usar
1. Digite o que você imagina (ex: "Botão neon azul").
2. Clique em **Materializar Ideia**.
3. Copie o código gerado ou veja o resultado no preview em tempo real.
